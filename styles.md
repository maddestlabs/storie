# Inline Formatting Demo

Welcome to the **enhanced** Storie engine with *inline formatting* support!

## Bold Text

You can make text **bold** by wrapping it in double asterisks.

This is **very important** information that stands out.

## Italic Text

Text can be *emphasized* with single asterisks or _underscores_.

This is *really cool* and _quite elegant_.

## Combined Formatting

You can even combine them: **bold and *italic* together**!

Or try *italic with **bold** inside*.

## Links

Links are displayed with underlines: [Click here](https://example.com)

You can link to documentation: [Nim Language](https://nim-lang.org)

Or reference resources: [Markdown Guide](https://www.markdownguide.org)

## Mixed Content

Here's a paragraph with **bold**, *italic*, and [linked](https://github.com) text all together. This demonstrates how the styling system handles multiple inline elements seamlessly.

The parser recognizes **complex *nested* formatting** and renders it properly with appropriate terminal styling attributes.

## Technical Details

- Links appear in **blue** with underlines
- Bold text uses the ANSI bold attribute
- Italic text uses the ANSI italic attribute
- All attributes can be combined

Press **Q** to exit.
